## 📌 About the Project

**FitHub CRM** is a full-stack fitness management system designed to handle everything from member onboarding to revenue tracking — all in one place.

This project is built with a strong focus on:

* ⚡ Performance
* 🔐 Security
* 🎯 User Experience
* 📊 Data-driven insights

---

## 🧩 Core Features

### 👥 Member Management

* Add / Update / Delete members
* Profile tracking with history
* Membership status monitoring

### 🧑‍🏫 Trainer Management

* Assign trainers to members
* Track trainer performance

### 📅 Smart Attendance System

* Daily check-in system
* Attendance logs & reports

### 💳 Subscription & Payments

* Plan creation (Monthly / Yearly)
* Payment tracking
* Auto renewal alerts

### 📊 Analytics Dashboard

* Revenue overview
* Member growth stats
* Attendance insights

## ⚙️ Getting Started

### 🔹 Clone Repo

```bash
git clone https://github.com/your-username/fithub-crm.git
cd fithub-crm
```

### 🔹 Install Dependencies

```bash
# Backend
cd server
npm install

# Frontend
cd client
npm install
```

### 🔹 Run Project

```bash
# Start backend
npm start

# Start frontend
npm start
```
